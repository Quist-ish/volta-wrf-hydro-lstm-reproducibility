# Zenodo Description

This record archives the reproducibility package for a hybrid WRF-Hydro-LSTM residual-correction framework applied to AMJJASO high-flow reconstruction in the regulated Volta River Basin. The package contains configuration files, dataset inventory, data availability statement, reproducibility checklist, script templates, processed domain-level files when available, and corrected supplementary outputs.

G-RUN ENSEMBLE is treated only as a runoff/reanalysis benchmark for residual construction and benchmark-relative comparison. GRDC, GloFAS, Global Flood Database, JRC Global Surface Water, and Sentinel-1 SAR are separate discharge or spatial evidence sources.
